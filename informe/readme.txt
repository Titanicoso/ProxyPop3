in order to propperly output a text file you need to install xml2rfc with:
  'sudo apt-get install xml2rfc'

to output to text file exectute:
  'xml2rfc informe.xml --text -u'

to edit the report i suggest using an IDE that checks the dtd in real time.
For more info on the base of this document: https://tools.ietf.org/html/rfc2629
